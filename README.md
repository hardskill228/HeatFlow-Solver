# HeatFlow-Solver
HeatFlow Solver — масштабований веб-сервіс для паралельного розв'язування задач теплопередачі (МСЕ). Архітектура включає Load Balancer, багатосерверну API та розподілену чергу завдань (Celery Workers), що забезпечує високу доступність, балансування навантаження та інформування клієнта про хід виконання через WebSockets.
